# CodeVerse 🚀

A full-stack coding platform where students can solve coding problems, submit solutions, and track their progress, while administrators can manage coding problems.

## ✨ Features

### Student
- User Registration & Login
- JWT Authentication
- View Coding Problems
- Solve & Submit Solutions
- My Submissions
- Leaderboard
- Profile Dashboard

### Admin
- Admin Dashboard
- Add Problems
- Edit Problems
- Delete Problems
- Manage Problems

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- JWT Authentication
- MySQL

## 📁 Project Structure

```
CodeVerse
├── client
└── server
```

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/pavanikandregula0808-droid/CodeVerse.git
```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
